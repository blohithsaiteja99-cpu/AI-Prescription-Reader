# AI Prescription Reader

AI Prescription Reader is a Flask-based web application that converts handwritten doctor prescriptions into digital text using OCR and image processing.

## Features
- Prescription image upload
- OCR text extraction
- Image preprocessing
- User-friendly interface

## Technologies
- Python
- Flask
- OpenCV
- EasyOCR
